936. По стакану, тату (ЖТ-ЛР-З)

1. По стакану, тату,
По стакану, мамо,
Як не дасиш по стакану,
То кинемо в яму.
Як не дасиш по стакану,
То кинемо в яму.
2. По стакану, по стакану,
По стакану рому,
Як не дасте по стакану,
Підемо додому.
Як не дасте по стакану,
Підемо додому.
3. По стакану, мамо,
По стакану, мамо,
Як не дасте по стакану,
Розкидаєм хату.
Як не дасте по стакану,
Розкидаєм хату.
